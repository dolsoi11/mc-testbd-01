# Project Blueprint

## Overview

This project is a simple lottery number generator. It allows users to generate a set of random lottery numbers.

## Current Implementation

### Features

*   Generate 6 random numbers between 1 and 45.
*   Display the generated numbers.

### Design

*   The application has a clean and simple user interface.
*   A "Generate Numbers" button initiates the number generation.
*   The generated numbers are displayed in a clear and readable format.
